# Склады продавца

Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Маркетплейс</strong>

  Узнать, как использовать методы в бизнес-кейсах, можно в <a href="https://dev.wildberries.ru/knowledge-base/articles/019d49a4-1320-71bb-9dac-8ba07e7177ce/rabota-s-tovarami">инструкции</a> по <strong>работе с товарами</strong>

  Узнать больше о складах продавца можно в <a href="https://seller.wildberries.ru/instructions/material/A-6?categoryId=2a88d2ed-45c4-4205-a540-7d0e87f0193d">справочном центре</a>

На складах продавца доступно управление [остатками товаров](./work-with-products#tag/Ostatki-na-skladah-prodavca).

Чтобы работать со складами продавца, вы можете:
  1. Получать [список складов WB](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1offices/get), если вы работаете по модели [FBS](./orders-fbs#tag/Zakazy-FBS) (Fulfillment by Seller). Это нужно, чтобы связывать склады WB и склады продавца при [создании](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1warehouses/post) и [редактировании](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1warehouses~1%7BwarehouseId%7D/put) склада.
  2. Получать [список складов продавца](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1warehouses/get), если работаете по модели продаж со склада продавца.
  2. [Получать](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1dbw~1warehouses~1%7BwarehouseId%7D~1contacts/get) и [обновлять](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1dbw~1warehouses~1%7BwarehouseId%7D~1contacts/put) список контактов складов модели Курьером WB (DBW).
  3. [Создавать](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1warehouses/post) склады продавца.
  4. [Обновлять](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1warehouses~1%7BwarehouseId%7D/put) склады продавца.
  5. [Удалять](./work-with-products#tag/Sklady-prodavca/paths/~1api~1v3~1warehouses~1%7BwarehouseId%7D/delete) склады продавца.
