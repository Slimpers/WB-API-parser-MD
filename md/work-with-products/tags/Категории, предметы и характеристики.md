Для [создания карточек товаров](./work-with-products#tag/Sozdanie-kartochek-tovarov) необходимо:
  1. Определить [родительскую категорию](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1object~1parent~1all/get), к которой будет относиться товар.
  2. Внутри категории выбрать [предмет](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1object~1all/get).
  3. Подобрать для каждого предмета [характеристики](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1object~1charcs~1%7BsubjectId%7D/get) товара. Характеристики [Цвет](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1directory~1colors/get), [Пол](work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1directory~1kinds/get), [Страна производства](work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1directory~1countries/get), [Сезон](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1directory~1seasons/get), [Ставка НДС](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1directory~1vat/get) и [ТНВЭД-код](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1content~1v2~1directory~1tnved/get) можно получить с помощью отдельных методов.
  4. Выбрать [бренд](./work-with-products#tag/Kategorii-predmety-i-harakteristiki/paths/~1api~1content~1v1~1brands/get) товара.
