# Цены и скидки

Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Цены и скидки</strong>

  Узнать, как использовать методы в бизнес-кейсах, можно в <a href="https://dev.wildberries.ru/knowledge-base/articles/019d49a4-1320-71bb-9dac-8ba07e7177ce/rabota-s-tovarami">инструкции</a> по <strong>работе с товарами</strong>

  Узнать больше о ценах и скидках можно в <a href="https://seller.wildberries.ru/instructions/ru/ru/subcategory/85fc88fa-a455-4df8-b530-bc67761edf84">справочном центре</a>

С помощью этих методов можно устанавливать [цены и скидки](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1upload~1task/post), в том числе [цены для размеров одного товара](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1upload~1task~1size/post) и [скидки WB Клуба](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1upload~1task~1club-discount/post).

Когда вы обновляете цены или скидки, данные по каким-то товарам могут не обновиться. Например, если вы передали неправильную цену или скидку. Проверяйте статус загрузки с помощью метода [состояния обработанной загрузки](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1history~1tasks/get).

Статусы загрузки:
  - `3` — обработана. В товарах нет ошибок, цены и скидки обновились.
  - `4` — отменена.
  - `5` — обработана, но в товарах есть ошибки. Ошибки можно получить с помощью метода [детализации обработанной загрузки](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1history~1goods~1task/get). Для товаров без ошибок цены и скидки обновились.
  - `6` — обработана, но во всех товарах есть ошибки. Ошибки можно получить с помощью метода [детализации обработанной загрузки](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1history~1goods~1task/get).

Если вы задаёте скидки в [календаре акций](./promotion#tag/Kalendar-akcij), загрузки попадают в обработку. Скидка применится в момент старта акции. У такой загрузки будет статус `1`, получить информацию о ней можно с помощью методов [состояния необработанной загрузки](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1buffer~1tasks/get) и [детализации необработанной загрузки](./work-with-products#tag/Ceny-i-skidki/paths/~1api~1v2~1buffer~1goods~1task/get).

  У загрузки не может быть статуса <code>2</code>.
