# Медиафайлы

Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Контент</strong>

  Узнать, как использовать методы в бизнес-кейсах, можно в <a href="https://dev.wildberries.ru/knowledge-base/articles/019d49a4-1320-71bb-9dac-8ba07e7177ce/rabota-s-tovarami">инструкции</a> по <strong>работе с товарами</strong>

Управлять медиафайлами в карточке товара можно двумя способами:
  1. Через прямую [загрузку одного медиафайла](./work-with-products#tag/Mediafajly/paths/~1content~1v3~1media~1file/post) в карточку товара.
  2. Через [перечисление ссылок на медиафайлы](./work-with-products#tag/Mediafajly/paths/~1content~1v3~1media~1save/post). В этом случае новые медиафайлы заменяют медиафайлы, уже добавленные в карточку товара.
