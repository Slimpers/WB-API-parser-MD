# Операции при приёмке

Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Аналитика</strong>

Чтобы получить отчёт об [операциях при приёмке](https://seller.wildberries.ru/analytics-reports/acceptance-report):
  1. [Создайте отчёт](./reports#tag/Operacii-pri-priyomke/paths/~1api~1v1~1acceptance_report/get).
  2. Дождитесь, когда отчёт будет готов. Вы можете [проверить статус](./reports#tag/Operacii-pri-priyomke/paths/~1api~1v1~1acceptance_report~1tasks~1%7Btask_id%7D~1status/get) готовности отчёта. Готовый отчёт хранится 2 часа.
  3. [Получите отчёт](./reports#tag/Operacii-pri-priyomke/paths/~1api~1v1~1acceptance_report~1tasks~1%7Btask_id%7D~1download/get).
