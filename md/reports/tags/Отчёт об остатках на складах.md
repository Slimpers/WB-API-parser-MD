# Отчёт об остатках на складах

Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Аналитика</strong>

  Узнать больше об отчётах по остаткам на складах можно в <a href="https://seller.wildberries.ru/instructions/material/A-244?categoryId=5f2162c5-069b-416d-a4e1-48da2a76e6b0">справочном центре</a>

Чтобы получить отчёт об [остатках на складах WB](https://seller.wildberries.ru/analytics-reports/warehouse-remains):
  1. [Создайте отчёт](./reports#tag/Otchyot-ob-ostatkah-na-skladah/paths/~1api~1v1~1warehouse_remains/get).
  2. Дождитесь, когда отчёт будет готов. Вы можете [проверить статус](./reports#tag/Otchyot-ob-ostatkah-na-skladah/paths/~1api~1v1~1warehouse_remains~1tasks~1%7Btask_id%7D~1status/get) готовности отчёта. Готовый отчёт хранится 2 часа.
  3. [Получите отчёт](./reports#tag/Otchyot-ob-ostatkah-na-skladah/paths/~1api~1v1~1warehouse_remains~1tasks~1%7Btask_id%7D~1download/get).
