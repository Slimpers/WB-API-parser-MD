# Управление пользователями продавца

Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Пользователи</strong>

С помощью этих методов вы можете:
  - [Создать приглашение для пользователя](./api-information#tag/Upravlenie-polzovatelyami-prodavca/paths/~1api~1v1~1invite/post) с доступом к профилю продавца
  - [Получить список активных или приглашённых пользователей продавца](./api-information#tag/Upravlenie-polzovatelyami-prodavca/paths/~1api~1v1~1users/get)
  - [Изменить права доступа пользователей](./api-information#tag/Upravlenie-polzovatelyami-prodavca/paths/~1api~1v1~1users~1access/put) к профилю продавца
  - [Закрыть доступ пользователю](./api-information#tag/Upravlenie-polzovatelyami-prodavca/paths/~1api~1v1~1user/delete) к профилю продавца

Управлять доступом пользователей можно только с токеном активного владельца профиля продавца.
