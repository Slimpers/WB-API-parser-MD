# История остатков

<div class="description_auth">
  Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Аналитика</strong>
</div>

<div class="description_important">
  Узнать больше об аналитике остатков можно в <a href="https://seller.wildberries.ru/instructions/ru/ru/material/stock-history-report">справочном центре</a>
</div>

Это информация из детализированной таблицы товаров и виджета детализации по регионам.<br></br>
Остатки в ответах данных методов — на текущий день.

<div class="description_important">
  Чтобы получать остатки по дням за период до 3 месяцев от текущей даты, используйте методы <a href='./analytics#tag/Analitika-prodavca-CSV'>Аналитика продавца CSV</a> — тип отчёта <code>STOCK_HISTORY_DAILY_CSV</code>
</div>

Методы получения [отчёта по статистике остатков](https://seller.wildberries.ru/content-analytics/history-remains):
  1. Текущих [остатков на складах WB](./analytics#tag/Istoriya-ostatkov/operation/postV1StocksReportWbWarehouses) по размерам
  2. Данных по таблице товаров с агрегацией по [группам](./analytics#tag/Istoriya-ostatkov/paths/~1api~1v2~1stocks-report~1products~1groups/post), [товарам](./analytics#tag/Istoriya-ostatkov/paths/~1api~1v2~1stocks-report~1products~1products/post), [размерам](./analytics#tag/Istoriya-ostatkov/paths/~1api~1v2~1stocks-report~1products~1sizes/post)
  3. Данных виджета **Статистика по регионам отгрузки** с детализацией по [складам](./analytics#tag/Istoriya-ostatkov/paths/~1api~1v2~1stocks-report~1offices/post)
