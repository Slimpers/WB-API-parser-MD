# Финансовые отчёты

<div class="description_auth">
  Для доступа к методам используйте <a href="./api-information#tag/Avtorizaciya/Kak-sozdat-personalnyj-bazovyj-ili-testovyj-token">токен</a> для категории <strong>Финансы</strong>
</div>

<div class="description_important">
  Узнать больше о финансовых отчётах можно в <a href="https://seller.wildberries.ru/instructions/ru/ru/subcategory/financial-reports">справочном центре</a>
</div>

Методы получения:
  1. [Списка](./financial-reports-and-accounting#tag/Finansovye-otchyoty/operation/postV1SalesReportsList) отчётов реализации и детализаций к отчётам по [ID](./financial-reports-and-accounting#tag/Finansovye-otchyoty/operation/postV1SalesReportsDetailedReportId) и за [период](./financial-reports-and-accounting#tag/Finansovye-otchyoty/operation/postV1SalesReportsDetailed)
  2. [Списка](./financial-reports-and-accounting#tag/Finansovye-otchyoty/operation/postV1AcquiringList) отчётов об издержках на приём платежей и детализаций к отчётам по [ID](./financial-reports-and-accounting#tag/Finansovye-otchyoty/operation/postV1AcquiringDetailedReportId) и за [период](./financial-reports-and-accounting#tag/Finansovye-otchyoty/operation/postV1AcquiringDetailed). Доступно только для продавцов из России
